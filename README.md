# Memory Game 🐱

## Descripción

**Memory Game** es un minijuego de memoria desarrollado en Unity donde el jugador controla un gato que recorre un tablero formado por bloques en el suelo que al saltar sobre un bloque, este revela una imagen oculta. 
El jugador debe memorizar la ubicación de las imágenes y encontrar todas las parejas antes de que el tiempo llegue a cero.

---

## Objetivo del juego

Encontrar las **8 parejas de imágenes iguales** ocultas en los bloques del tablero antes de que finalice el tiempo establecido.

---

## Mecánica de juego

- El escenario contiene una cuadrícula de **9 bloques (8 parejas)**.
- Cada bloque inicia mostrando un signo de interrogación (?).
- El jugador controla un gato chibi que puede desplazarse por el escenario.
- Al saltar sobre un bloque serevela la imagen que contiene.
- Si dos bloques consecutivos muestran la misma imagen:
  - Permanecen descubiertos.
- Si las imágenes son diferentes:
  - Se vuelven a ocultar después de unos segundos.
- El jugador debe memorizar la ubicación de las imágenes para completar todas las parejas.

---

## Condiciones de victoria

- Descubrir correctamente las **8 parejas** antes de que el temporizador llegue a cero.

---

## Condiciones de derrota

- El tiempo se agota sin haber encontrado todas las parejas.

---

## Escenario

El minijuego se desarrolla en un:

- Ambiente cálido de primavera.
- Vegetación decorativa.
- Cuadrícula de bloques distribuidos como un tablero de memoria.

---

## Personaje

El jugador controla un gato
